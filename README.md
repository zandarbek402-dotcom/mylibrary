# MyLibrary

MyLibrary – кітапханаға арналған қарапайым ақпараттық жүйе.
Жоба Advanced Backend & DevOps курсы аясында командалық жоба ретінде әзірленуде.

## Мақсаты
Кітаптарды, авторларды және пайдаланушыларды басқаруға арналған веб-қосымша әзірлеу.
Жүйе REST API, дерекқор, контейнеризация және CI/CD қолданады.

## Негізгі мүмкіндіктер
- Пайдаланушы тіркелуі және кіруі (JWT)
- Пайдаланушы рөлдері: USER, ADMIN
- Кітаптарды басқару (CRUD)
- Іздеу, сүзгілеу, сұрыптау, пагинация
- Әкімші панелі
- Логирование және мониторинг
- Docker және CI/CD

## Технологиялар
- Backend: Django REST Framework
- Database: PostgreSQL
- Frontend: React
- DevOps: Docker, GitHub Actions
- Monitoring: Prometheus, Grafana
